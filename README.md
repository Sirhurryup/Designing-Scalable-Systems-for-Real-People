# Designing-Scalable-Systems-for-Real-People

## Welcome

Welcome to my cloud engineering portfolio.

This repository documents my transition from principal investigator and professor to cloud engineer through a growing collection of consulting engagements, technical case studies, and engineering reflections.

Every project is built around a simple idea.

Technology should solve real problems for real people.

I design and build systems that do not just work. They make sense to the people using them. 

With a background as a principal investigator in cancer survivorship research, I approach cloud engineering through system thinking, human-centered design, and evidence-based decision making. My goal is not simply to build infrastrucuture, but to design systems that businesses can trust and people can confidently use.
