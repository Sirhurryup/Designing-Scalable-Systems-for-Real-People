# The SirhurryUp Consulting Playbook

> **Technology changes. Principles endure.**

The SirhurryUp Consulting Playbook documents the principles, habits, and lessons earned through real-world cloud consulting engagements.

This is not a collection of AWS notes.

AWS documentation already exists.

This playbook exists to capture the way I think, investigate, communicate, design, and grow as an engineer. Every principle was earned through an engagement, a mistake, a difficult question, or a lesson that changed the way I approach my work.

As technology evolves, these principles should remain relevant because they focus on people, process, and disciplined engineering rather than individual tools or cloud services.
